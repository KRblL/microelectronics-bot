# microelectronics-bot

Telegram-бот для автоматизированного подбора аналогов микроэлектронных компонентов по парт-номеру.

## Возможности

- прием парт-номера через Telegram;
- поиск и загрузка технической документации из каталога AllDatasheet;
- извлечение и нормализация текста из PDF-документации;
- определение основного компонента и его ключевых характеристик через Grok API;
- подбор аналогов с использованием Grok API и web-search;
- формирование текстового ответа и сравнительной XLSX-таблицы.

## Настройка

```bash
cp example.env .env
```

В `.env` необходимо указать токен Telegram-бота и ключ Grok API.

## Запуск через Docker Compose

```bash
docker compose up -d --build
```

## Локальный запуск

```bash
pip install -r requirements.txt
python -m bot.bot
```

## Команды

- `/start` — запуск бота;
- `/help` — краткая справка;
- текстовое сообщение с парт-номером — запуск обработки компонента.
